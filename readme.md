# onenet 组件功能预处理


这个仓库给出了onenet 复杂网络组件库预处理阶段的代码。
模型包括

* 动态网络模型
    * ER 网络
    * 构型网络
    * 小世界可导航网络
    * self **阿波罗网络**
    * BA 网络模型
* 社团网络模型
    * caveman gaph
    * LRF 模型
* 加权网络模型
    * self **BBV 模型**
    * self **流驱动模型**
    
其中 表示有self 加粗的是自己复现的，其他的是在networkx里面有开源代码的


依赖python package

* networkx
* matplotlib
* numpy

感谢 包帮忙复现的阿波罗网络。
